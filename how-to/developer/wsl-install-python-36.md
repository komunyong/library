
## Install python 3.6 in WSL

```
sudo add-apt-repository ppa:jonathonf/python-3.6
sudo apt-get update
sudo apt-get install python3.6
```

**Note**: For Ubuntu 16.04, you can just `sudo apt-get install python3.6`
