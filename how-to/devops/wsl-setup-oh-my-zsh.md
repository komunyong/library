# Make WSL (Window Subsystem Linux) nice looking

## Process

```sh
# Install Zsh shell
sudo apt-get install zsh

# Install git
sudo apt-get install git

# Install oh-my-zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
